Simple Window

This sample opens a simple window with an URL inside.

FramesManager framesManager = new FramesManagerFactory().createFramesManager(); ... GInternalFrame frame = framesManager.newFrame(); frame.setWidth(600); frame.setHeight(312); frame.setStyle("alphacube"); frame.setTitle("Window with an URL inside"); frame.setUrl("site/demo/simplescenarii.html"); frame.showCenter(false);
